WebConnectorApp
=============

Application to showcase Scrapper Javaservice.


# Purpose
The process of extracting content from a remote web page is known as HTML Scraping. 
The Kony Scraper Tool is used to scrape data from an external web page. 
The data extracted with the Scraper is available to the Mobile Application 
as it would be if it were coming from a web service or another data connection type. The Scrapper Service app shows how “AboutUS” content from http://www.kony.com using scraper tool and displayed on the UI.

# Supported platforms:
**Mobile**
 * iPhone
 * Android
